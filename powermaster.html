How to Install

  • STEP 1: Select the correct download from above that matches your computer's operating system architecture.
  • STEP 2: Wait for the ZIP folder to download from your browser and, once finished, open its location in the Windows File Explorer.
  • STEP 3: Right click on the ZIP folder and select, "Extract All..." and follow the on-screen prompts in the window that subsequently opens.
  • STEP 4: Once the ZIP folder has finished extracting, open the extracted folder that should be named, "PowerMaster-<version you downloaded>-Installer-<architecture you downloaded>."
  • STEP 5: There should be a file named, "PowerMaster-<version you downloaded>-<architecture you downloaded>-Installer.exe." If you want to install PowerMaster with the default install location of, "C:\ProgramFiles" or "C:\ProgramFiles (x86)," (if you download the x86 build) right click on the executable file and run it as administrator to prevent an error from being thrown. However, if you plan to install PowerMaster into a location like "%APPDATA%" that does not require administrator access to write to, you can simply double click the EXE file as it is and be sure to select a path as so during setup.
  • STEP 6: When attempting to open the EXE file as mentioned earlier, you might have a Microsoft Defender SmartScreen window appear. To continue with the setup, click, "More info" and then, "Run Anyway" to successfully launch the installer.
  • STEP 7: Once you have successfully opened the EXE file, simply proceed with the installation by following the instructions on-screen. After clicking next on the initial install view, the next screen where it requires that the, "Name," and, "Company," fields be inputted will only prerequisite that you enter text in the, "Name," text box.
  • STEP 8: Next, ensure that the, "Typical Setup," option is selected before proceeding to the subsequent view where you can either leave the default install location or modify it accordingly.
  • STEP 9: Here, you have the option of renaming the start menu folder and choosing whether it will be displayed for all users or just yourself. The folder will contain the main program executable as well as the uninstall EXE file for convenience. You can leave the default options or modify them to your liking and press, "Next," to get to the install confirmation screen and then click, "Next" once more to begin the download.
  • STEP 10: Once the installer has finished configuring PowerMaster, you will encounter this screen. Simply uncheck, "Run PowerMaster now," (as it will NOT eventually do this) and press the, "Finish," button.
  • FINALLY: Now all you have to do is access your Start Menu, open the, "PowerMaster," folder as shown below (or the folder with whatever name you gave it during setup), and select, "PowerMaster," to launch the app. There is also an uninstallation program as well that you can run as administrator to ultimately remove PowerMaster from your computer in the future.

Features
  • Switch between the Power Saver and High Performance power modes with a click of a button.
  • Enable the option to keep PowerMaster minimized in your system tray for easy access and quick actions.
  • Let PowerMaster automatically select the Power Saver power mode whenever you are on battery power.
  • Monitor battery-relevant information like its overall status, time for full charge and discharge, and whether your laptop is plugged in or not.
  • View PowerMaster's included Power Widget for efficient and simplified access to battery and power information.
  • Select between light and dark mode for the app to display.
